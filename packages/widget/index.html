Lightning Widget

Accept Lightning payments on any website through a Nostr HTTP tunnel. No server exposed to the internet — your node stays behind the tunnel.

1. Run the tunnel agent on your server

The widget talks to your Lightning node (e.g. phoenixd) through a Nostr HTTP tunnel. The tunnel agent runs alongside your node and forwards requests over Nostr relays.

docker compose up -d   # starts tunnel agent + phoenixd auth proxy

2. Add the widget to your site

<script src="https://unpkg.com/@nostr-http-tunnel/widget"></script>
<lightning-widget
  npub="npub1..."
  relays="wss://relay.damus.io,wss://nos.lol"
></lightning-widget>

Set npub to your tunnel agent's public key. The widget connects to the relays, sends encrypted requests, and displays a QR code — all client-side, no backend needed.
